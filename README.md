# 📚 University Central Library Analytics Dashboard

## 📌 Project Overview

The **University Central Library Analytics Dashboard** is a data analytics and visualization project designed to analyze university library usage patterns, student engagement behavior, study area occupancy, and book borrowing trends using interactive dashboards and analytical techniques.

The project transforms raw institutional-style library datasets into meaningful insights through:

* data preprocessing,
* feature engineering,
* exploratory data analysis,
* dashboard visualization,
* and institutional reporting.

The system provides an interactive analytical environment for understanding:

* department-wise library activity,
* borrowing behavior,
* study duration trends,
* peak occupancy hours,
* and resource utilization patterns.

---

# 🎯 Project Objectives

* Analyze library usage patterns across departments
* Identify popular book categories
* Track monthly and temporal borrowing trends
* Analyze student study duration behavior
* Develop interactive dashboards for institutional insights
* Demonstrate a complete data analytics workflow
